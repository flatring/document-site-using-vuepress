      # Draw.io Integration

      1. https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio
      2. これとPlantUMLがあれば大概の絵が書けます
      3. 例えばGoogleCloudの構成図が書きたければ↓からアイコンが使えます
        1. 公式のアイコンと図のサンプル

      # Markdown All in One

      1. https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one
      2. tableのフォーマットが便利(Alt + Shift + f)
      3. 箇条書きの際にインクリメントされると却って面倒なので 1. で固定にする
        1. settings.json
          // Markdown All in One
"markdown.extension.list.indentationSize": "inherit",
"markdown.extension.orderedList.marker": "one",

      # markdownlint

      1. https://marketplace.visualstudio.com/items?itemName=DavidAnson.vscode-markdownlint
      2. markdownのlint
      3. rootに置いたlintファイル .markdownlint.yml のルールに従って問題点を指摘しててくれる
      4. lintさせない行がある場合には↓のように記述する
        <!-- markdownlint-disable MD037 -->
deliberate space * in * emphasis
<!-- markdownlint-enable MD037 -->

        2
