        # Markdown All in One

        1. https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one (opens new window)
        2. tableのフォーマットが便利(Alt + Shift + f)
        3. 箇条書きの際にインクリメントされると却って面倒なので 1. で固定にする
          1. settings.json
            // Markdown All in One
"markdown.extension.list.indentationSize": "inherit",
"markdown.extension.orderedList.marker": "one",

        # markdownlint

        1. https://marketplace.visualstudio.com/items?itemName=DavidAnson.vscode-markdownlint (opens new window)
        2. markdownのlint
        3. rootに置いたlintファイル .markdownlint.yml のルールに従って問題点を指摘しててくれる
        4. lintさせない行がある場合には↓のように記述する
          <!-- markdownlint-disable MD037 -->
deliberate space * in * emphasis
<!-- markdownlint-enable MD037 -->

          2
          3
