      # 自分の名前登録する

      1. プロジェクトに合わせたルールで
      2. 例: ローマ字で「名字 + (半角スペース) + 名前」とする
        # 設定
git config --global user.name "suzuki taro"
# 確認
git config -l global | grep user.name
